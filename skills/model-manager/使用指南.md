## 🚀 **快速开始**

### **基本使用**

```bash
# 进入目录
cd skills/model-manager

# 规划任务
python3 manage_models.py plan "你的任务描述"

# 查看模型列表（需要网络）
python3 manage_models.py list
```

### **配置模式**

```bash
# 最大节省模式
python3 manage_models.py enable cheap

# 平衡模式
python3 manage_models.py enable balanced

# 最佳质量模式
python3 manage_models.py enable quality
```

## ⚙️ **高级配置**

### **自定义路由规则**

创建 `~/.openclaw/model-routing.json`：

```json
{
  "patterns": {
    "translation": ["gemini-2.0-flash", "gpt-4o-mini"],
    "coding": ["claude-3.5-sonnet", "gpt-4o"],
    "analysis": ["gpt-4o", "claude-3.5-sonnet"]
  },
  "fallbacks": ["gpt-4o-mini"],
  "budget_limit": 50.00
}
```

### **成本监控**

```bash
# 设置预算提醒
python3 manage_models.py monitor --budget 100 --alert-at 80%
```

### **性能分析**

```bash
# 生成报告
python3 manage_models.py report --days 30 --export csv
```

## 📋 **命令参考**

### **plan - 任务规划**
```bash
python3 manage_models.py plan "任务描述"
```
**输出：**
- 任务分类
- 推荐模型
- 成本节省

### **list - 模型列表**
```bash
python3 manage_models.py list
```
**输出：**
- 可用模型
- 实时价格
- 性能排名

### **enable - 启用模式**
```bash
python3 manage_models.py enable <mode>
```
**模式：**
- `cheap` - 最大节省
- `balanced` - 平衡
- `quality` - 最佳质量

## ⚠️ **已知问题**

### **网络连接**
**问题：** `list` 命令 SSL 错误
**解决：**
1. 检查网络连接
2. 使用 VPN
3. 使用 `plan` 命令（不受影响）

### **配置文件**
**问题：** 配置未生效
**解决：**
```bash
# 手动重载配置
openclaw reset
```
