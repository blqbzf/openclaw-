# 诺兰时光 WoW 登录器

## 🎯 技术栈

**Avalonia UI** - 跨平台 .NET 桌面应用框架

### 为什么选择 Avalonia?
- ✅ **体积小** - 10-20MB (vs Electron 150MB+)
- ✅ **性能好** - 原生应用，不是浏览器包装
- ✅ **跨平台** - Windows/macOS/Linux 一次编写到处运行
- ✅ **无依赖** - 不需要浏览器引擎
- ✅ **开发快** - C# + XAML，类似 WPF

### 功能列表
- [x] 客户端目录选择
- [x] realmlist 自动修复
- [x] 文件验证
- [x] 游戏启动

### 开发环境
- .NET 8.0+
- Avalonia 11.0+

### 构建产物
- Windows: 10-15MB
- macOS: 12-18MB
- Linux: 8-12MB
