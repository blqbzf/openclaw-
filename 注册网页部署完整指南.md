## ⚠️ 常见问题

### Q1: 访问提示"无法连接"？

**A:** 检查防火墙是否开放5000端口

```bash
# 检查防火墙
firewall-cmd --list-ports

# 开放5000端口
firewall-cmd --permanent --add-port=5000/tcp
firewall-cmd --reload
```

### Q2: 注册提示"数据库连接失败"？

**A:** 检查MySQL是否运行，密码是否正确

```bash
# 检查MySQL
systemctl status mysqld

# 测试连接
mysql -u acore -pP1WoW2026! acore_auth
```

### Q3: 页面显示但注册失败？

**A:** 查看日志

```bash
tail -50 /root/wow-register/register.log
```

常见原因：
- MySQL密码错误 → 修改 `register_server.py` 中的 `DB_CONFIG`
- 用户名重复 → 提示用户换用户名
- 邮箱重复 → 提示用户换邮箱
